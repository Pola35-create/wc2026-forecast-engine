# WC2026 Forecast Engine

A data-driven football forecasting project focused on the FIFA World Cup 2026.

The goal of this project is to build a complete tournament prediction system capable of forecasting:

* Match outcomes (win/draw/loss probabilities)
* Exact match score prediction (e.g. 2-1, 1-1)
* Expected goals (xG-style modeling per team)
* Match event probabilties (e.g. penalty awarded, red card likelihood)
* Group standings
* Knockout stage progression
* Tournament winner
* Awards winners (e.g. Golden Boot, Golden Glove, etc...)

The system will continuously update as new information becomes available, including:

* Match results
* Team form
* Injuries
* Suspensions
* Squad announcements
* Expected lineups

## Project Objectives

### Phase 1

Build a probabilistic match prediction engine using team ratings, historical performance, and goal distribution modeling.

### Phase 2

Build a Monte Carlo tournament simulator capable of running thousands of World Cup simulations.

### Phase 3

Develop player-level forecasting models for individual awards.

### Phase 4

Introduce automated updates and dynamic team ratings.

### Phase 5

Evaluate machine learning approaches to improve forecasting accuracy.
